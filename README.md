# Health Insurance Reimbursement Platform

##  Overview
This project is a web application developed using Angular to track health insurance reimbursements.

##  Features
- Dashboard for monitoring reimbursements
- Insured management
- Reimbursement tracking system
- Admin and client interfaces
- Dynamic routing and lazy loading

## Technologies
- Angular
- DevExtreme
- HTML, CSS
